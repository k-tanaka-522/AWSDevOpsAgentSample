# R-04 UI/UXコンセプト

## 1. デザインキーワード
- **Focus (集中)**: ユーザーが「今やるべきこと」だけに集中できるノイズの少ない画面。
- **Flow (没入)**: 引っかかりのないスムーズな操作性、アニメーション。
- **Modern Pro (専門性)**: プロフェッショナルなツールとしての信頼感、ダーク基調のクールな配色。

## 3. デザインガイドライン

### 3.1 カラーパレット
| 色名 | カラーコード | 用途 |
|---|---|---|
| **Background** | `#1E1E1E` | メイン背景色（ダークグレー） |
| **Surface** | `#2D2D2D` | カード、サイドバー背景 |
| **Primary** | `#0078D4` | アクションボタン、強調 |
| **Danger** | `#E81123` | 期限切れ、削除、超高優先度 |
| **Warning** | `#FFB900` | 高優先度 |
| **Text Main** | `#FFFFFF` | 主要テキスト |
| **Text Sub** | `#AAAAAA` | 補足テキスト |

### 3.2 タイポグラフィ
- **Font Family**: `Segoe UI` (標準), `Cascadia Code` (数値を扱う箇所)
- **Base Size**: 14px

### 3.3 コンポーネントルール
- **角丸 (Radius)**: ボタンやカードは `CornerRadius="4"` または `8` を採用。
- **影 (Shadow)**: 深度を持たせるため、DropShadowEffect を控えめに使用。
- **アニメーション**: リストの並び替え時、タスク完了時に 0.2s - 0.3s 程度のイージングアニメーションを入れる。

## 4. 画面イメージ概要
### メイン画面
3カラム構成を基本とする。
1. **Nav**: 左端。アイコン＋テキスト。常時表示。
2. **List**: 中央。最も広い領域。カード型のタスクリスト。
3. **Detail**: 右端。選択タスクの詳細。何もないときは「今日の統計（消化数など）」を表示し、スペースを有効活用する。
