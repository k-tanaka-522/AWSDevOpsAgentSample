

# 概要
オンプレからAWSの移行を検討しているので実現可能かPOCで構築する。

# POCで作るもの
簡単なCRUDなwebアプリ(フロントnode.js api gateway,API(ECS+python),RDS)を作り、x-ray,cloudwatch sns (HTMLメール)、AWS DevOps agent(最近発表されたAWSのエージェント)を使い、品質管理、監視、AIによる１次調査を検証する。ほか、監視、品質、セキュリティ、監査で必要なAWSサービスがあれば、提案してください。

あくまで私の範囲はインフラ領域でアプリケーションは、サンプル程度の品質で構わない。
ただし x-rayによるレスポンス追跡の評価をしたいのであえて応答の遅いapiを作り、検証シナリオをつくって検証する。


以下は、検証ア終わったら追加で検証。
